## 5. Project Overview

I developed a full-stack ERP Management System using MongoDB, Express.js, React, and Node.js.
The application provides core ERP workflows such as product and inventory management, customer and supplier records, sales order handling, invoice management, and dashboard analytics. The project also includes secure role-based authentication using JWT.

## 6. Key Features Implemented

- JWT-based authentication and authorization
- Role-based user access (admin, manager, employee)
- Product management with stock tracking
- Customer management and supplier management
- Sales order management
- Invoice generation and invoice listing
- Dashboard with business insights and KPIs
- Sample production data seeded in MongoDB Atlas

## 7. Technology Stack

- Frontend: React, React Router, Axios
- Backend: Node.js, Express.js, Mongoose
- Database: MongoDB Atlas
- Authentication/Security: JWT, bcrypt

## 8. Local Setup Steps

1. Clone the repository.
2. Configure backend environment variables in `.env`.
3. Configure frontend environment variables in `.env`.
4. Install dependencies in both backend and frontend using `npm install`.
5. Seed demo data using `npm run seed` in backend.
6. Start backend using `npm run dev`.
7. Start frontend using `npm start`.
